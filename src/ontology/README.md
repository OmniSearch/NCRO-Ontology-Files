Do not edit chebi-imports.owl - it is generated by ontofox. To update, add to ontofox-inputs/chebi.txt, cd ../tools, make imports.

Do not add to imports.owl - its contents will be replaced by ontofox-generated imports. If you want to import a term, do it analogously to how the ChEBI terms are imported.

Do not commit changes to catalog-v001.xml - Protege has a tendency to write it too often and screw it up. To get rid of protege changes use the command 'git checkout catalog-v001.xml'
